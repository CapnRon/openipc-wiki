Wyze Cam V2
-----------

- Manufacturer: Wyze
- ODM : Hualai
- SoC: Ingenic T20X
- Sensor: JX-F22 or JX-F23
- Flash: GD25Q128CSIG [c8 40 18 c8 40]
- WiFi: Realtek 8189FTV


### UART

    -------------
       o  o  o   |
       G  T  R   |
                 |


### Confire Wi-Fi

```
fw_setenv wlandev rtl8189ftv-t20-wyze-v2
fw_setenv wlanssid mygloriousnetwork
fw_setenv wlanpass IaMaKo0LhaZk3r
```
